OrthoAid

While doing our research on tooth meta tagging, we found a windows-based app developed by the Shahid Beheshti University of Medical Sciences (SBMU) in collaboration with the University of British Columbia (UBC), whose purpose is to help orthodontists analyse pre and post operative 3D dental study models. This app can display 3D tooth models in ply format and is written in C#. The app has MIT license, and it solves the problem of displaying ply file format, so we plan to modify their source code to be used in our app. Their source code can be found here. Client meeting about TWES 2.0

Our client proposed to have an online meeting so that he can provide further explanation on the Tooth Wear Evaluation System (TWES 2.0). From the meeting, we learnt more about the purpose of using TWES 2.0, and the goal our app should achieve. TWES 2.0 has two parts, for the first part, Tooth Wear Screening, the AI model should be able to segment the tooth model into six sextant and output the highest grade of each sextant. As for the second part, Tooth Wear Status, the AI model should be able to output the highest grade of each tooth. After having the tooth wear grade, together with the patient information inputted by dentist when using the app, the AI model should be able to analyse the cause of tooth wear (mechanical tooth wear, chemical tooth wear, pathological tooth wear).
