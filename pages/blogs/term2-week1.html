General structure for deep learning model

During the holidays, our client sent us some tooth model data that he bought online, we created a general structure for the deep learning model using PointNet. PointNet is a neural network that directly consumes point clouds and well respects the permutation invariance of points in the input, it is very suitable for deep learning on point sets for 3D classification and segmentation. However, we are still waiting for labels of tooth wear grade from our client.

Develop frontend using PySide and Open3D

Although we plan to use C# for the frontend app development and modify the source code of OrthoAid, we found out that OrthoAid is not able to properly display ply file encoded in binary format. As the data supplied by our client is encoded in binary format, we decided to give up on using C# and develop the frontend using Python instead. We chose Python because we found a very useful Python library, Open3D that is able to display 3D images in PLY format.
