Moving database to cloud

Since one of the team members found a way to maintain the pre-trained deep-learning model on the Azure virtual machine, we used a similar method to host our SQLite database on Azure. Python Requests library is used to send and receive data between the client and the server. Besides, we decided to store the tooth model ply files as binary file instead of storing file paths. It took us a few days to get the binary files displaying with Open3D. The main challenge we faced is that some data is lost in the process of sending to the server, causing the tooth models to be displayed in a wrong position.
