Deployment Manual

This deployment manual will explain how to deploy the application to a local computer. As this is a Windows desktop software, users will need to download the .exe file to start the application.

Prerequesites

As we are using the Win32GUI Library in our application, the application can only be opened in a Windows environment. Hence you need to have a Windows laptop or set up a Windows environment on Mac/Linux to run Wear3D.

Run Application

Cloud Azure

Note: As we host our database on the Azure Virtual Machine, you need to connect to our server before being able to use the application. This part can be done on Windows/Mac/Linux laptop.

Step 1: Open the terminal (on Mac/Linux) or Command Prompt (on Windows) on your local computer.

Step 2: Use SSH to connect to the cloud server, and the password for the virtual machine is: Team10Toothwear! 

ssh team10@20.127.200.67

Enter the password when you are prompted after connected to the cloud.

Step 3: Once you are connected to the remote virtual machine, you will need to navigate to the directory where the app.py file is located. You can navigate to the directory by using the following command: 

cd tooth-wear-dl/back-end/app/

Step 4: Run the back-end code by using the following command: 

python app.py

Now, the back-end preparation is done. You can now open Wear3D (after downloading it following the instructions below) on your local computer and enjoy the app!

Download and Run Wear3D

Note: This part must be done on a Windows laptop.

Step 1: Navigate to our github website using this link.

Step 2: Click on the Wear3D release on the right hand side.

Step 3: Click on the "executable.zip" file under Assets to download.

Step 4: When the download is completed, extract the zip file by right-clicking on executable.zip and select "Extract All..."

Step 5: Navigate into the "app" folder in the extracted "executable" folder and double-click "main.exe" to run Wear3D.
