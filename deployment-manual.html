Deployment Manual

This deployment manual will explain how to deploy the application to a local computer. As this Windows desktop software, users will need to download the .exe file to start the application. This deployment mannual assumes that the user has had a GitHub account and has git in the environment.

Prerequesites

Note: As we are using the Win32GUI Library in our application, the application can only be opened in a Windows environment.

Git: Download and install Git on your local computer. You can download Git from the following link: "https://git-scm.com/book/en/v2/Getting-Started-Installing-Git"

Build and Run

Step 1: Open the "Command Prompt" app:

Click the Start button in the lower-left corner of your desktop to open your Start menu. Type and search "cmd" on the Start menu. Command Prompt should show up at the top of the search results. Click Command Prompt on the Start menu. This will open a new Command Prompt window.

Step 2: Clone the project to your local device: 

git clone https://github.com/TilenLS/COMP0016-2022-10.git

Step 3: Run the app from the terminal: 

start main.exe

Alternatively, you can also run the app from the file explorer:
1: Open the file explorer and navigate to the directory where the app is located.
2: Double click the main.exe file to run the app.

Cloud Azure

Step 1: Open a terminal on your local computer

Step 2: Use SSH to connect to the cloud server, and the password for the virtual machine is: Team10Toothwear! 

                
ssh team10@20.127.200.67

Enter the password when you are prompted after connected to the cloud.

Step 3: Once you are connected to the remote virtual machine, you will need to navigate to the directory where the app.py file is located. You can navigate the directory by using the following command: 

                
cd tooth-wear-dl/back-end/app/

Step 4: Run the back-end code by using the following command: 

                
python app.py

Now, the back-end preparation is done. You can now open the software on your local computer and enjoy the app!
