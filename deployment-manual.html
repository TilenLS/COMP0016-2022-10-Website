Deployment Manual

This deployment manual will explain how to deploy the application on your local computer. As this is a Windows desktop software, users will need to download the .exe file to start the application.

Prerequesites

As we are using the Win32GUI Library in our application, the application can only be opened in a Windows environment. Hence you need to have a Windows laptop or set up a Windows environment on Mac/Linux to run Wear3D. To run the application, you will need to have the following installed on your computer (or need to know how to install these):

  • Git
  • Python environment: Anaconda / Miniconda / python
  • pip

Run Application

Cloud Azure

Note: As we host our database on the Azure Virtual Machine, you need to connect to our server before being able to use the application. This part can be done on Windows/Mac/Linux laptop.
You can also skip this step and run the software directly for testing, which will connect to our team's server by default.

Step 1: Go to Azure website and create a virtual machine. Detailed guideline can be found here. Make sure to take note of the username, password, and your vm ip address for the virtual machine.

Step 2: Open the terminal (on Mac/Linux) or Command Prompt (on Windows) on your local computer.

Step 3: Use SSH to connect to the cloud server. The command is as follows: 

ssh username@your-vm-ip-address

Step 4: Once you are connected to the remote virtual machine, you will need to clone our repository. Note: this manual assumes that you have git and a python environment (eg. miniconda) installed on your VM. 

git clone https://github.com/HongruiTang/tooth-wear-dl.git

Step 5: Dowload related packages from the file requirements.txt: 

cd tooth-wear-dl
pip install -r requirements.txt

Step 6: Navigate to the directory where the app.py file is located. 

cd back-end/app/

Step 7: Run the back-end code by using the following command: 

python app.py

   If you want to run the app back-end code continuously on the server, you can use the following command: 

nohup python app.py &

Now, the back-end preparation is done. You can now open Wear3D (after downloading it following the instructions below) on your local computer and enjoy the app!

For software downloading, please refer to the guidance on this page. However, you can also download the source code from our GitHub repository.
