Deployment Manual

This deployment manual will explain how to deploy the application on your local computer. As this is a Windows desktop software, users will need to download the .exe file to start the application.

Prerequesites

As we are using the Win32GUI library in our application, Wear3D can only be opened in a Windows environment. Hence you need to have a Windows laptop or set up a Windows environment on Mac/Linux to run Wear3D.

If you want to run the application using our source code instead of the executable file, you will need to have the following installed on your computer (or need to know how to install these):

  • Git
  • Python environment: Anaconda / Miniconda / Python
  • pip

Run Application

Cloud Azure

Note: As we host our database on the Azure Virtual Machine, you need to connect to our server before being able to use the application. This part can be done on Windows/Mac/Linux laptop.
You can also skip this step and run the software directly for testing, which will connect to our team's server by default.


Step 1: Go to Azure website and create a virtual machine. Detailed guideline can be found here. Make sure to take note of the username, password, and your vm ip address for the virtual machine.

Step 2: Open the terminal (on Mac/Linux) or Command Prompt (on Windows) on your local computer.

Step 3: Use SSH to connect to the cloud server. The command is as follows: 

ssh username@your-vm-ip-address

Step 4: Once you are connected to the remote virtual machine, you will need to clone our repository. Note: this manual assumes that you have git and a python environment (eg. miniconda) installed on your VM. 

git clone https://github.com/HongruiTang/tooth-wear-dl.git

Step 5: Download related packages from the file requirements.txt: 

cd tooth-wear-dl
pip install -r requirements.txt

Step 6: Navigate to the directory where the app.py file is located. 

cd back-end/app/

Step 7: Run the back-end code by using the following command: 

python app.py

   If you want to run the app back-end code continuously on the server, you can use the following command: 

nohup python app.py &

Now, the back-end preparation is done. You can now open Wear3D (after downloading it following the instructions below) on your local computer and enjoy the app!

Download and Run Wear3D

Note: The following needs to be done on a Windows environment.

Download Executable File

The easiest way to use Wear3D is to download the zip file on our GitHub Release. Please follow the instructions below to do so.


Step 1: Navigate to our github website using this link.

Step 2: Click on the Wear3D release on the right hand side.

Step 3: Click on the "executable.zip" file under Assets to download.

Step 4: When the download is completed, extract the zip file by right-clicking on executable.zip and select "Extract All..."

Step 5: Navigate into the "app" folder in the extracted "executable" folder and double-click "Wear3D.exe" to run Wear3D.

(Optional) Step 6: If the "Windows protected your PC" warning pop up when running the app, click on "More Info" as shown below:

(Optional) Step 7: Then click on "Run anyway" to run the application.

There is an "Example Scans" folder together with Wear3D.exe in the "app" directory. The "Example Scans" folder consists of upper jaw scans, lower jaw scans, and sextant scans available for testing purposes.

Download Source Code

Alternatively, you can choose to download the source code of our application from our Github repository. Please follow the instructions below to do so.


Step 1: Open a terminal and clone our GitHub repository: 

git clone https://github.com/TilenLS/Wear3D.git

Step 2: Download related packages from the file requirements.txt: 

pip install -r requirements.txt

Step 3: Navigate to the directory where the main.py file is located. 

cd src/cdss

Step 4: Run the software 

python main.py
