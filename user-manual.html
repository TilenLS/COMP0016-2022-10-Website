User Manual

Log-in Page

This is your standard and simple login page to access the rest of the app, for those who are already registered. If not press the register button to make an account. (Register page explained below)

  1. Enter your email and password in the boxes indicated.
  2. Then press "Sign In" to sign in and continue to the home page.

Register Page

  1. Enter a valid email and correct password. Don't forget to confirm your password.
  2. Then press "Register" to create your account, then move on to sign in.

Home Page

  1. When you login you will first see the home page. On the left sidebar there will be a navigation panel to access all app pages.
  2. The main table on the home page is the database of all the patients. (You can click on a patient's ID to directly view their tooth scans. Clicking on the header of each column will sort the table correspondingly).
  3. Click the add patient button to add a new patient. This will cause a sidebar on the right to appear.
  4. Here you can input all the patient details along with the patient tooth scans. Once you are done press `add patient`. (You can select upper jaw, lower jaw, and sextant scans from the "Example Scans" folder that is located in the same directory as Wear3D.exe. The need to select a sextant scan is because our tooth wear grading deep learning model is built with sextant scans. By uploading sextant scans while you add a new patient, the most accurate prediction about tooth wear grade will be provided when run analysis.)
  5. Scroll with your mouse or using the scroll bar on the bottom to see the rest of the patient data.
  1. At the end of the patient table there is a delete button to remove a patient.
  2. Make sure you want to delete the selected patient and confirm.

Viewing Page

  1. In the viewing page, a patient's tooth scans can be seen and interacted with via using the cursor.
  2. The right sider-bar contains functionalities such as temporarily hiding a specific scan.
  3. The right side-bar also contains functionalites to add different scans manually or reset the viewpoint.

Results Page

  1. In the results page you simply enter a selected patient id and press run analysis. You will see the prediction load in on top after some time.
  2. The bottom will display the treatment plan suggestion.
